# Django Weather App

A Django-based Weather Application built using Python and Django.

## Features

- User Registration
- User Login
- User Logout
- Weather Search
- OpenWeatherMap API Integration
- Protected Dashboard

## Technologies Used

- Python
- Django
- HTML
- CSS
- SQLite
- OpenWeatherMap API

## Run Project

```bash
python manage.py runserver
```

Open:

```
http://127.0.0.1:8000/
```
